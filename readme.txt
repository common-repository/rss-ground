== Description ==

If you are having troubles connecting third-party scripts to your WordPress's XML-RPC, this plugin is for you.

This plugin offers alternative XML-RPC file that is not blocked by your hosting provider or security plugins.

It was originally developed for RSSGround.com users who set up automated posting campaigns for their WordPress blogs but it can be used by anybody else.
